# Shared Workflow Ritual

1. Save all changed files in your editor, then `cd /Users/cdmxx/Documents/design-awards-inspiration`.
2. Run `git status` to see the latest tree and confirm the base commit.
3. Commit meaningful changes with clear messages; document any manual steps that the other tool should know.
4. Keep `NEXT_STEPS.md` and `HANDOFF_TEMPLATE.md` up to date before ending your session.
5. Signal the next collaborator to re-read the disk (close/reopen files) after the repo moves or new files land.
