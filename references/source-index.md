# Source Index

Use these official sources when the bundled 50-site catalog is too narrow or when you need fresher references.

All links below were checked on March 20, 2026.

## Primary Seed Source

- Awwwards `Sites of the Day` archive:
  [https://www.awwwards.com/websites/sites_of_the_day/](https://www.awwwards.com/websites/sites_of_the_day/)
  Use for high-signal award winners with embedded tag metadata that can be normalized into categories, visual styles, and technology cues.

## Secondary Expansion Sources

- CSS Design Awards `Website of the Year 2025` nominees:
  [https://www.cssdesignawards.com/woty2025/](https://www.cssdesignawards.com/woty2025/)
  Use when you want extra 2025 references with category labels, short writeups, studios, and judge scores.

- SiteInspire:
  [https://www.siteinspire.com/](https://www.siteinspire.com/)
  Use when the user wants broader inspiration by type, style, or subject, even when the site is not specifically tied to an award.

## Expansion Rule

Prefer the local dataset first. Expand only when:

- the user explicitly asks for more references
- the user wants a niche not covered by the bundled catalog
- the user wants live or newer sources
